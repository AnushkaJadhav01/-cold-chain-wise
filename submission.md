# Submission Package

## Problem Statement
Modern global supply chains manage millions of concurrent shipments across highly complex and volatile transportation networks. Disruptions such as sudden weather events, infrastructure failures, and operational bottlenecks are often detected only after delivery timelines have already been compromised, leading to spoilage, increased costs, and customer dissatisfaction.

## Solution Overview
We built **Cold‑Chain‑Wise**, a web‑based dashboard powered by React, TypeScript, and a digital‑twin **What‑If Simulator**. The simulator lets logistics planners instantly tweak delay, temperature, and distance parameters and see a real‑time risk score with AI‑style recommendations. This enables proactive route adjustments before disruptions cascade, improving resilience and reducing spoilage.

Key features:
- Interactive sliders for delay (0‑10 h), temperature (0‑15 °C), distance (50‑500 km).
- Real‑time risk calculation (0‑100 %) with green/yellow/red visual cues.
- Contextual recommendations & Gemini‑style explanations.
- Seamless integration into the existing dashboard without altering layout or styling.
- Deployable on any cloud platform (Vercel, Netlify, AWS Amplify, etc.).
